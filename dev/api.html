# SteerableConvolutions.AbstractRepresentation — Type.
julia
abstract type AbstractRepresentation

Abstract group representation.

A representation

ρ:G→Cd×d

maps group elements g to matrices ρ(g) of size d×d. The representation must be such that

ρ(gh)=ρ(g)ρ(h)∀(g,h)∈G2.

# SteerableConvolutions.IrreducibleRepresentation — Type.
julia
struct IrreducibleRepresentation{G, A} <: AbstractRepresentation

Irreducible group representation ("irrep").

Fields

  • group: Group.

  • id: Irrep ID.

  • mat: Irrep matrix constructor. Can be called as mat(g).

  • type: Irrep type, can be 'R', 'C', or 'Q'.

# SteerableConvolutions.Representation — Type.

struct Representation{A} <: AbstractRepresentation

General group representation. It is stored as a direct sum of irreps with a basis change.

Fields

  • basis: Change of basis matrix.
