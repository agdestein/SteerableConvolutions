Equivariance ​

Introduction to GSpaces, FieldTypes, and FiberFields using the SteerableConvolutions.jl package.

julia
using SteerableConvolutions
using CairoMakie

GSpaces ​

Create 2-dimensional GSpace with cyclic group of order 4.

julia
G = CyclicGroup(4)
gspace = GSpace(G, 2)
GSpace{CyclicGroup}(CyclicGroup(4), 2)

Field types ​

Create a FieldType from a list of representations.

julia
representations = [irrep(G, 0), irrep(G, 1)]
fieldtype = FieldType(gspace, representations)

Fiber fields ​

Since our fieldtype contains one scalar field (from irrep(G, 0)) and one 2D-vector field (from irrep(G, 1)), we need to create a feature field with three channels. We wrap it in a FiberField from a field type and data.

julia
n = 50
grid = LinRange(0, 1, n + 1)
mask = @. (1 - grid^2) * (1 - (grid')^2)
x = zeros(n + 1, n + 1, 3)
@. x[:, :, 1] = sinpi(10 * grid) * mask
@. x[:, :, 2] = sinpi(2 * grid) * cospi(3 * grid') * mask
@. x[:, :, 3] = -cospi(2 * grid) * sinpi(3 * grid') * mask
field = FiberField(fieldtype, x)

Plot the all fields

julia
function plot(field)
    (; x) = field
    s = @. sqrt(field.x[:, :, 2]^2 + field.x[:, :, 3]^2)
    color = s[:] ./ maximum(s)
    limits = (0, 1, 0, 1)
    fig = Figure(; size = (800, 400))
    heatmap(fig[1, 1], grid, grid, x[:, :, 1]; axis = (; limits, title = "Scalar field"))
    arrows(
        fig[1, 2],
        grid,
        grid,
        x[:, :, 2],
        x[:, :, 3];
        lengthscale = 0.1,
        color,
        axis = (; limits, title = "Vector field"),
    )
    fig
end

Equivariance ​

A function f mapping one fiber field u with representation ρin to another fiber field of representation ρout is said to be G-equivariant if

f(ρin(g)u)=ρout(g)f(u)

for all g∈G.

For example, consider ρin to be the representation from above, and ρout to be the trivial representation:

julia
fieldtype_out = FieldType(gspace, [irrep(gspace.group, 0)])
FieldType{GSpace{CyclicGroup}, Vector{IrreducibleRepresentation{CyclicGroup, Int64, SteerableConvolutions.var"#11#12"{Int64}}}}(GSpace{CyclicGroup}(CyclicGroup(4), 2), IrreducibleRepresentation{CyclicGroup, Int64, SteerableConvolutions.var"#11#12"{Int64}}[IrreducibleRepresentation{CyclicGroup, Int64, SteerableConvolutions.var"#11#12"{Int64}}(CyclicGroup(4), 0, SteerableConvolutions.var"#11#12"{Int64}(0), 'R')])

The following function is not equivariant:

julia
function notequivariant(u)
    (; x) = u
    y = x[:, :, 2:2] + x[:, :, 3:3]
    FiberField(fieldtype_out, y)
end

julia
a = g * notequivariant(field)
b = notequivariant(g * field)
a.x ≈ b.x
false

is false. The following function is equivariant:

julia
function norm2(u)
    (; x) = u
    y = @. x[:, :, 2:2]^2 + x[:, :, 3:3]^2
    FiberField(fieldtype_out, y)
end

b = norm2(g * field)
